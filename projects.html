Here there are some of my most interesting projects:

Haventrip

Haventrip is a web app where you can search apartments for your vacations! This was the Boolean's final project. With Haventrip you can search, create, delete, modify or sponsorize your apartments with a simple and user-friendly interface. If you register yourself into this website you'll have at your service tools on your properties like create, delete, modify and sponsorize.

'Let the journey begun...Choose Haventrip!'

MixTales...

Mixtales is a group project where we've created a showcase site for a club. Using Axios we called the data from the database and showed them dinamically with a paginator. It was not only about showing data, it was about telling stories from the cocktails point of view so we created an environment that could tell these stories.

'Don't follow your tale...join MixTales.'

Clone DC-COMICS

Simple clone of the DC-COMICS website showing dynamic data on cards. The back-end created with php sends the data to the front created with Vuejs.

'You Either Die A Hero, Or You Live Long Enough To See Yourself Become A Villain.'

BoolzApp

Simple clone of the famous WhatsApp-Web with some pre-defined messages programmed with JS that shows up when you write something.

'No fantastic quote here...just sends messages.'
