Some of my most interesting projects:

Haventrip

Boolean final project, Haventrip, is a web app that can be used to search for accomodations for your vacations! Thanks to a registration system and a user-friendly interface you'll be able to create, delete, modify and sponsorize your apartments. The user's validation system prevents other users from accessing your data.

'Let the journey begin...Choose Haventrip!'

MixTales...

Mixtales is ashowcase site for a club. It was created in team using Axios to call the data from the database to show them in a dynamic way by a paginator. It wasn't just about showing data, but telling the stories behind each cocktail in an inviting display.

'Don't follow your tail...join MixTales.'

Clone DC-COMICS

Clone-site of the DC-COMICS website showing dynamic data on cards. The back-end created with php sends the data to the front created with Vuejs.

'You Either Die A Hero, Or You Live Long Enough To See Yourself Become A Villain.'
